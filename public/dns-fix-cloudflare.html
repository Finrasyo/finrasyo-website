🔧 DNS Düzeltme Rehberi

❌ Hata: DNS verification failed

Sorun: "finrasyo.com doesn't appear to be served by Netlify"

Sebep: Cloudflare DNS ayarları yanlış veya proxy aktif

🚀 ADIM ADIM ÇÖZÜM

  1. Cloudflare Dashboard'a gidin
  2. finrasyo.com domain'ini seçin
  3. DNS → Records sekmesine gidin
  4. "www" kaydını bulun
  5. "Edit" butonuna tıklayın
  6. Aşağıdaki ayarları yapın:
DOĞRU CLOUDFLARE AYARLARI: Type: CNAME Name: www Target: [NETLIFY-URL].netlify.app TTL: Auto Proxy status: DNS only (GRİ BULUT - ÖNEMLİ!) YANLIŞ AYAR: Proxy status: Proxied (TURUNCU BULUT)

⚠️ Kritik Noktalar

  • Proxy KAPALI olmalı: Gri bulut simgesi
  • Target doğru olmalı: Netlify URL'iniz
  • Type CNAME olmalı: A record değil
  • Name "www" olmalı: @ veya finrasyo.com değil

📋 Netlify URL Nasıl Bulunur

  1. Netlify dashboard → Sites
  2. Site'inizi bulun
  3. Site adına tıklayın
  4. URL'i kopyalayın (örnek: amazing-butterfly-12345.netlify.app)
  5. Bu URL'i Cloudflare'de Target olarak kullanın

✅ Ayarları Yaptıktan Sonra

  1. "Save" butonuna tıklayın
  2. 5-10 dakika bekleyin (DNS propagation)
  3. Netlify'e geri dönün
  4. "Retry DNS verification" butonuna tıklayın
  5. Başarılı olursa SSL otomatik başlayacak

🎯 Hemen Yapın

1. Cloudflare → DNS → www kaydını edit edin

2. Proxy'yi kapatın (gri bulut)

3. Target'ı Netlify URL yapın

4. 10 dakika bekleyin

5. Netlify'de "Retry DNS verification"
