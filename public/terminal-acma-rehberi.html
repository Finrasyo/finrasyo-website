💻 Terminal/Command Prompt Açma

🪟 WINDOWS

Yöntem 1: Windows Tuşu + R

  • Windows + R tuşlarına basın
  • "cmd" yazıp Enter tuşuna basın

Yöntem 2: Başlat Menüsü

  • Sol alt köşedeki Windows simgesine tıklayın
  • "cmd" veya "command prompt" yazın
  • Çıkan sonuca tıklayın

Yöntem 3: Proje Klasöründe

  • Proje klasörünü File Explorer'da açın
  • Adres çubuğuna "cmd" yazıp Enter basın
  • Terminal direkt o klasörde açılır

🍎 MAC

Yöntem 1: Spotlight

  • Cmd + Space tuşlarına basın
  • "Terminal" yazıp Enter tuşuna basın

Yöntem 2: Launchpad

  • Launchpad'i açın
  • "Other" klasörüne gidin
  • "Terminal" uygulamasına tıklayın

Yöntem 3: Finder

  • Applications → Utilities → Terminal

🐧 LINUX

Klavye Kısayolu

  • Ctrl + Alt + T (çoğu dağıtımda)

Applications Menüsü

  • Activities/Applications açın
  • "Terminal" arayın

✅ Terminal Açıldıktan Sonra

Proje klasörüne gitmek için:

  • Windows: cd C:\path\to\your\project
  • Mac/Linux: cd /path/to/your/project
  • Kolay yol: Proje klasörünü terminale sürükleyin
