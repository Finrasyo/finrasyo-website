🖥️ Shell Nasıl Kullanılır

Komutları Shell'e yazıp Enter'a basın

📝 Basit Adımlar

1. Shell'i Gördünüz

Shell açık ve şu şekilde görünüyor:

~ $ █

Bu cursor (imleç) - komut yazabileceğinizi gösterir.

2. İlk Komutu Yazın

İlk komut: cd /home/runner/workspace

Nasıl: Bu komutu kelime kelime yazın

~ $ cd /home/runner/workspace█

Sonra: Enter tuşuna basın!

3. Enter'a Bastıktan Sonra

Komut çalışacak ve yeni satır görünecek:

~ $ cd /home/runner/workspace
/home/runner/workspace $ █

Görüyorsunuz? Klasör değişti! Artık ikinci komutu yazabilirsiniz.

🚀 Tüm Komutlar Sırayla

Komut 1: Klasöre Git

Yazın: cd /home/runner/workspace

Enter'a basın

~ $ cd /home/runner/workspace
/home/runner/workspace $ █

Komut 2: Git Başlat

Yazın: git init

Enter'a basın

/home/runner/workspace $ git init
Initialized empty Git repository in /home/runner/workspace/.git/
/home/runner/workspace $ █

Komut 3: GitHub Bağla

ÖNEMLİ: KULLANICI_ADI yerine gerçek GitHub kullanıcı adınızı yazın!

Yazın: git remote add origin https://github.com/KULLANICI_ADI/finrasyo.git

Enter'a basın

/home/runner/workspace $ git remote add origin https://github.com/mehmet123/finrasyo.git
/home/runner/workspace $ █

Komut 4: Dosyaları Hazırla

Yazın: git add .

Enter'a basın

/home/runner/workspace $ git add .
/home/runner/workspace $ █

Bu komut sessizce çalışır, çıktı vermeyebilir.

Komut 5: Commit Oluştur

Yazın: git commit -m "Initial commit: FinRasyo platform"

Enter'a basın

/home/runner/workspace $ git commit -m "Initial commit: FinRasyo platform"
[main (root-commit) abc1234] Initial commit: FinRasyo platform
25 files changed, 1500 insertions(+)
/home/runner/workspace $ █

Komut 6: Branch Ayarla

Yazın: git branch -M main

Enter'a basın

/home/runner/workspace $ git branch -M main
/home/runner/workspace $ █

Komut 7: GitHub'a Yükle

Yazın: git push -u origin main

Enter'a basın

/home/runner/workspace $ git push -u origin main
Username for 'https://github.com': █

Bu noktada GitHub kullanıcı adınızı yazın ve Enter'a basın!

⚠️ Authentication (Kimlik Doğrulama)

git push komutunda GitHub iki şey isteyecek:

  1. Username: GitHub kullanıcı adınız
  2. Password: Personal Access Token (GitHub şifreniz DEĞİL!)

Personal Access Token nasıl alınır:

  1. GitHub → Settings → Developer settings
  2. Personal access tokens → Tokens (classic)
  3. Generate new token → Classic
  4. Permissions: "repo" seçin
  5. Token'ı kopyalayın

Her komuttan sonra Enter'a basmayı unutmayın!

Komut → Enter → Sonuç → Yeni komut → Enter

Shell'e yazın ve Enter'a basın!
