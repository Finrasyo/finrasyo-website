🔄 Site Silme ve Yeniden Upload

SSL sorununu çözmek için temiz başlangıç

İndirilecek dosya: finrasyo-build-fixed.tar.gz Boyut: 1.8MB İçerik: dist/public/ + netlify.toml (React Router fix)
1️⃣ ADIM: Yeni Dosyayı İndirin
  1. Replit Files panelinde finrasyo-build-fixed.tar.gz dosyasını bulun
  2. Dosyaya sağ tıklayın → Download
  3. Masaüstüne indirin
  4. Dosyayı çift tıklayarak açın
  5. dist/public/ klasörünü çıkarın

✅ Kontrol: Çıkarılan klasörde index.html, assets/ ve netlify.toml olmalı

2️⃣ ADIM: Mevcut Site'i Silin
  1. netlify.com → Dashboard'a gidin
  2. Mevcut finrasyo site'ini bulun
  3. Site'in üzerine tıklayın
  4. Site settings butonuna tıklayın
  5. En aşağı scroll edin
  6. "Danger zone" bölümünü bulun
  7. "Delete this site" butonuna tıklayın
  8. Site adını yazarak onaylayın

⚠️ Uyarı: Site silinince www.finrasyo.com bağlantısı kesilir. Yeniden bağlamanız gerekir.

3️⃣ ADIM: Yeni Site Upload
  1. Netlify ana sayfa → "Add new site"
  2. "Deploy manually" seçin
  3. Mavi drag & drop alanını görün
  4. dist/public/ klasörünü sürükleyin
  5. Upload başlayacak (2-3 dakika)
  6. Yeni otomatik URL alacaksınız

📋 Not: Upload sırasında progress bar göreceksiniz

4️⃣ ADIM: Domain Tekrar Bağlayın
  1. Yeni site'in dashboard'ına gidin
  2. Site settings → Domain management
  3. "Add custom domain"
  4. "www.finrasyo.com" yazın
  5. "Yes, add domain" onaylayın
  6. DNS bilgilerini kopyalayın
  7. Cloudflare'de www record'unu güncelleyin
5️⃣ ADIM: Cloudflare DNS Güncelleme
  1. Cloudflare → DNS → Records
  2. "www" kaydını bulun
  3. "Edit" tıklayın
  4. Target: Yeni Netlify URL'i yazın (örnek: new-amazing-site-12345.netlify.app)
  5. Proxy status: DNS only (gri bulut)
  6. "Save" tıklayın

🎯 Beklenen Sonuç (15-30 dakika sonra)

  • ✅ www.finrasyo.com açılır
  • ✅ SSL sertifikası otomatik oluşur
  • ✅ Navigation linkler çalışır (netlify.toml sayesinde)
  • ✅ Hakkımızda, İletişim, Nasıl Çalışır sayfaları açılır

🚨 Önemli Notlar

  • DNS propagation: 15-30 dakika sürebilir
  • SSL oluşma: Domain aktif olduktan sonra otomatik
  • Test yöntemi: Önce Netlify URL, sonra www.finrasyo.com
  • Sorun olursa: 24 saat bekleyin veya Netlify support'a yazın

🚀 Başlayın!

1. finrasyo-build-fixed.tar.gz indirin

2. Eski site'i silin

3. Yeni upload yapın

4. Domain bağlayın
