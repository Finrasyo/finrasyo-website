⚠️ Cloudflare Netlify Hataları

❌ HATA 1: "Invalid target hostname"

Sebep: Netlify URL formatı yanlış

✅ ÇÖZÜM:

Doğru format kullanın:

YANLIŞ: https://site-name.netlify.app DOĞRU: site-name.netlify.app YANLIŞ: www.site-name.netlify.app DOĞRU: site-name.netlify.app

Sadece domain adı, https:// veya www olmadan

❌ HATA 2: "CNAME target cannot be an IP address"

Sebep: IP adresi girmeye çalışıyorsunuz

✅ ÇÖZÜM:

  • IP adresi değil, Netlify domain kullanın
  • Netlify dashboard'da site URL'inizi bulun
  • .netlify.app uzantılı adresi kullanın

❌ HATA 3: "Target hostname already exists"

Sebep: Aynı domain başka yerde kullanılıyor

✅ ÇÖZÜM:

  1. Mevcut "www" kaydını silin
  2. 5 dakika bekleyin
  3. Yeni CNAME kaydı oluşturun
  4. Name: www, Target: netlify-url

❌ HATA 4: "Cannot resolve target hostname"

Sebep: Netlify URL yanlış yazılmış

✅ ÇÖZÜM:

Netlify URL'i doğru şekilde kopyalayın:

  1. Netlify → Sites → Site'inize tıklayın
  2. Site URL'ini kopyalayın
  3. .netlify.app ile bittiğinden emin olun
  4. Typo olmadığından emin olun

📋 Netlify URL Nasıl Bulunur

  1. Netlify.com → Dashboard
  2. Sites sekmesine gidin
  3. Site'inizin adına tıklayın
  4. URL'i kopyalayın (örnek: happy-einstein-123456.netlify.app)
  5. Bu URL'i Cloudflare'de Target olarak kullanın

🎯 Doğru Cloudflare Ayarı

Type: CNAME Name: www Target: [site-name].netlify.app Proxy: DNS only (gri bulut)
