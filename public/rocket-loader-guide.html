🚀 Rocket Loader Kapatma Rehberi

⚠️ Problem: Rocket Loader JavaScript'i geciktirir ve React navigation'unu bozar. Bot Fight Mode kapalıysa, muhtemelen Rocket Loader soruna neden oluyor.

1Cloudflare Dashboard'a Giriş

Tarayıcınızda dash.cloudflare.com adresine gidin

https://dash.cloudflare.com

Hesabınızla giriş yapın ve finrasyo.com domain'ini seçin

2Speed Menüsüne Gitme

Sol menüden Speed sekmesine tıklayın

Speed → Optimization

Ardından Optimization alt menüsünü seçin

3Rocket Loader Bulma

Sayfada Rocket Loader bölümünü bulun

Not: Rocket Loader genellikle "Web Optimization" veya "Performance" bölümünde yer alır

4Rocket Loader Kapatma

Rocket Loader'ın yanındaki toggle switch'i OFF konumuna getirin

Önemli: Rocket Loader OFF olduğunda genellikle gri renkte görünür

5Değişiklikleri Kaydetme

Değişiklikleri kaydetmek için Save butonuna tıklayın

✅ "Settings saved successfully" mesajı görünecek

6Cache Temizleme

Sol menüden Caching sekmesine gidin

Caching → Configuration

Purge Everything butonuna tıklayın ve onaylayın

⏱️ Bekleme Süresi: Rocket Loader kapatıldıktan sonra 5-10 dakika bekleyin ve cache temizleyin.

🧪 Test Alanı

10 dakika sonra aşağıdaki linklerle test edin:

🏠 Ana Sayfa 📄 Hakkımızda 📧 İletişim ⚙️ Nasıl Çalışır

🎉 Başarı Kontrolü

Eğer linkler tıklandığında sayfalar açılıyorsa:

  • ✅ Rocket Loader sorunu çözüldü
  • ✅ JavaScript navigation çalışıyor
  • ✅ Site normal durumda

🔄 Hala Çalışmıyorsa

Rocket Loader da kapatmak yeterli olmazsa:

  1. Mirage kapatın (Speed → Optimization)
  2. Polish kapatın (Speed → Optimization)
  3. Page Rule ekleyin (Rules → Page Rules)
  4. Development Mode açın (geçici)

📋 Page Rule Alternatifi

Rules → Page Rules menüsünden yeni rule ekleyin:

URL: www.finrasyo.com/*
Settings:
- Rocket Loader: Off
- Cache Level: Bypass
- Security Level: Essentially Off
