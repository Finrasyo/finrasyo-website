🔧 CNAME Invalid Hatası

❌ Hata: "Content for CNAME record is invalid"

Sebep: Netlify URL'i yanlış formatta girdiniz

Çözüm: Doğru format kullanmanız gerekiyor

YANLIŞ FORMATLAR: ❌ https://site-name.netlify.app ❌ http://site-name.netlify.app ❌ www.site-name.netlify.app ❌ site-name.netlify.app/ DOĞRU FORMAT: ✅ site-name.netlify.app

✅ ADIM ADIM ÇÖZÜM

  1. Netlify dashboard'a gidin
  2. Sites sekmesinde site'inizi bulun
  3. Site URL'ini kopyalayın
  4. URL'den sadece domain kısmını alın
  5. Cloudflare'de bu domain'i Target olarak girin

📋 Netlify URL Kopyalama

Netlify'de görünen URL örneği:

https://amazing-butterfly-123456.netlify.app

Cloudflare'de Target kutusuna yazacağınız:

amazing-butterfly-123456.netlify.app

Yani: https:// ve sonundaki / işaretlerini kaldırın

🎯 Cloudflare'de Doğru Ayar

Type: CNAME Name: www Content/Target: [site-name].netlify.app Proxy status: DNS only (gri bulut) TTL: Auto

🚀 Şimdi Yapın

1. Netlify dashboard → Sites → URL kopyalayın

2. URL'den https:// ve sondaki / kaldırın

3. Cloudflare'de Target kutusuna yapıştırın

4. Save tıklayın

Netlify URL'inizi doğru formatta girmeyi deneyin!
