🔐 SSL Sertifikası Doğrulama

✅ ŞU AN YAPMANIZ GEREKEN

  1. Netlify'de "Verify DNS configuration" butonuna tıklayın
  2. Netlify DNS kayıtlarını kontrol edecek
  3. Cloudflare ayarlarınızı doğrulayacak
  4. Sorun varsa size bildirecek

⚠️ Yaygın DNS Sorunları

Eğer DNS doğrulama başarısız olursa:

  • Proxy aktif: Cloudflare'de gri bulut olduğundan emin olun
  • Yanlış CNAME: Target'ın doğru Netlify URL olduğunu kontrol edin
  • DNS propagation: 15-30 dakika beklemek gerekebilir
  • Cache sorunu: Cloudflare cache temizleyin
Cloudflare DNS Kontrol: - Type: CNAME - Name: www - Target: [your-site].netlify.app - Proxy: DNS only (gri bulut)

🎯 DNS Doğrulama Başarılı Olursa

  • SSL sertifikası otomatik oluşacak (5-15 dakika)
  • "HTTPS certificate provisioned" mesajı görünecek
  • www.finrasyo.com güvenli şekilde açılacak
  • Navigation tamamen çalışacak

📋 Şimdi Yapın

1. "Verify DNS configuration" butonuna tıklayın

2. Sonucu bekleyin (2-3 dakika)

3. Başarılıysa SSL otomatik başlayacak

4. Başarısızsa DNS ayarlarını kontrol edin
