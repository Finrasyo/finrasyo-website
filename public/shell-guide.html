2

Shell Tab'ı Görünmüyorsa

Bazen Shell tab'ı görünmez. Bu durumda:

  • + (Plus) butonuna tıklayın (tab'ların yanında)
  • "Shell" seçeneğini bulup tıklayın
  • Yeni Shell tab'ı açılacak
3

Alternatif: Tools Menüsü

Başka bir yöntem:

  1. Sol sidebar'da "Tools" ikonuna tıklayın
  2. "Shell" seçeneğini bulun ve tıklayın
  3. Shell açılacak

✅ Shell Açıldığında Göreceğiniz

~ $ █

Bu görünümü gördüğünüz anda hazırsınız! Komutları yazabilirsiniz.

🔧 Git Komutları

4

Komutları Sırayla Çalıştırın

Shell açıldıktan sonra bu komutları birer birer yazın:

# 1. Proje klasörüne git cd /home/runner/workspace # 2. Git başlat git init # 3. GitHub repository bağla (KULLANICI_ADI değiştir) git remote add origin https://github.com/KULLANICI_ADI/finrasyo.git # 4. Dosyaları hazırla git add . # 5. Commit oluştur git commit -m "Initial commit: FinRasyo platform" # 6. Ana branch ayarla git branch -M main # 7. GitHub'a yükle git push -u origin main

⚠️ Önemli Notlar

  • KULLANICI_ADI yerine GitHub kullanıcı adınızı yazın
  • GitHub username ve password isteyecek
  • Password yerine Personal Access Token kullanın
  • Her komutun tamamlanmasını bekleyin

🔑 GitHub Authentication

5

Personal Access Token

git push komutunda GitHub şifre isteyecek:

  1. GitHub → Settings → Developer settings
  2. Personal access tokens → Tokens (classic)
  3. Generate new token
  4. repo permissions seç
  5. Token'ı kopyala ve password olarak kullan
