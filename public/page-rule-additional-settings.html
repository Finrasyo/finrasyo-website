🔧 Ek Ayarlar Ekleyin

Mevcut Page Rule'a ek ayarlar eklemek için:

  1. Page Rule'un yanındaki Edit (kalem) ikonuna tıklayın
  2. Aşağıdaki ek ayarları ekleyin:
Security Level: Essentially Off
Rocket Loader: Off
Mirage: Off
Polish: Off

💾 Kaydet ve Cache Temizle

  1. Save and Deploy butonuna tıklayın
  2. Sol menüden Caching → Configuration gidin
  3. Purge Everything butonuna tıklayın
  4. Onaylayın ve 30 saniye bekleyin
