🛡️ Bot Fight Mode Kapatma Rehberi

⚠️ Önemli: Bot Fight Mode, JavaScript'i bloke ederek normal kullanıcı etkileşimlerini engelliyor. Kapatmak güvenlidir ve site performansını artırır.

1Cloudflare Dashboard'a Giriş

Tarayıcınızda dash.cloudflare.com adresine gidin

https://dash.cloudflare.com

Cloudflare hesabınızla giriş yapın

2Domain Seçimi

Ana sayfada finrasyo.com domain'ini bulun ve tıklayın

[ Domain listesinde finrasyo.com'u bulup tıklayın ]

3Security Menüsüne Gitme

Sol menüden Security sekmesine tıklayın

Security → Settings

Ardından Settings alt menüsünü seçin

4Bot Fight Mode Bulma

Sayfada aşağı kaydırarak Bot Fight Mode bölümünü bulun

[ Bot Fight Mode: ON/OFF toggle switch ]

Genellikle sayfanın ortalarında yer alır

5Bot Fight Mode Kapatma

Bot Fight Mode'un yanındaki toggle switch'i OFF konumuna getirin

Önemli: Switch'in OFF konumunda olduğundan emin olun (genellikle gri renk)

6Değişiklikleri Kaydetme

Sayfanın en altında veya üstünde Save butonuna tıklayın

✅ Değişiklikler kaydedildi mesajı görünecek
⏱️ Bekleme Süresi: Değişiklikler 5-10 dakika içinde etkili olacaktır. Sabırlı olun.

🧪 Test Alanı

10 dakika sonra aşağıdaki linklerle test edin:

🏠 Ana Sayfa 📄 Hakkımızda 📧 İletişim ⚙️ Nasıl Çalışır

🎉 Başarı Kontrolü

Eğer yukarıdaki linkler tıklandığında sayfalar açılıyorsa:

  • ✅ Navigation sorunu çözüldü
  • ✅ Site normal çalışır durumda
  • ✅ Kullanıcı deneyimi düzeldi

🔄 Alternatif Çözümler

Bot Fight Mode kapatmak yeterli olmazsa:

  1. Rocket Loader da kapatın (Speed → Optimization)
  2. Page Rule ekleyin (Rules → Page Rules)
  3. Cache temizleyin (Caching → Purge Everything)
