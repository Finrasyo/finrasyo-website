🚨 Cloudflare Navigation Sorunu ve Çözümü

❌ Sorun:

Cloudflare proxy, JavaScript tabanlı tüm navigation metodlarını bloke ediyor:

  • React Link component'leri çalışmıyor
  • JavaScript onClick event'leri çalışmıyor
  • window.location.href çalışmıyor
  • Sadece direkt URL yazarak sayfalara erişilebiliyor

✅ Geçici Çözümler:

  1. Direkt URL kullanın:
    • www.finrasyo.com/about
    • www.finrasyo.com/how-it-works
    • www.finrasyo.com/contact
  2. HTML Form Navigation: Pure HTML form submit kullanıyoruz
  3. Meta Refresh: JavaScript olmadan sayfa yönlendirme
  4. Iframe Navigation: Sayfa içi navigation

🔧 Kalıcı Çözüm Adımları:

  1. Cloudflare Dashboard'a gidin:
    • dash.cloudflare.com
    • finrasyo.com domain'ini seçin
  2. Security → WAF:
    • JavaScript execution kurallarını kontrol edin
    • Bot Fight Mode'u devre dışı bırakın
  3. Speed → Optimization:
    • Rocket Loader'ı devre dışı bırakın
    • Mirage'ı devre dışı bırakın
  4. Caching → Configuration:
    • Browser Cache TTL'yi azaltın
    • Cache Everything'i devre dışı bırakın
  5. Page Rules ekleyin:
    URL: www.finrasyo.com/* Settings: - Cache Level: Bypass - Rocket Loader: Off - Mirage: Off - Security Level: Essentially Off

🚀 Alternatif Çözüm - Subdomain:

Ana domain'de sorun devam ederse:

  1. app.finrasyo.com subdomain'i oluşturun
  2. Cloudflare proxy'sini bypass edin (DNS Only)
  3. Tüm navigation'u subdomain'e yönlendirin
