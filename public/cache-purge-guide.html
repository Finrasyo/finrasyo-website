🧹 Cache Temizleme Rehberi

🎯 Amaç: Page Rule aktif olduktan sonra cache temizleyerek değişikliklerin hemen etkili olmasını sağlamak.

1Caching Menüsünü Bulma

Cloudflare Dashboard'da sol menüden Caching sekmesini arayın

Sol menü → Caching
[ Caching menüsü genellikle Speed'in altında yer alır ]

2Configuration Alt Menüsü

Caching'e tıkladıktan sonra Configuration alt menüsünü seçin

Caching → Configuration

Bu sayfada cache ayarları ve temizleme seçenekleri bulunur

3Purge Everything Butonu

Sayfada Purge Everything butonunu bulun

Konum: Genellikle sayfanın üst kısmında, mavi veya turuncu renkte bir buton
[ Purge Everything - Tüm cache'i temizler ]

4Cache Temizleme

Purge Everything butonuna tıklayın ve onaylayın

✅ "Cache purged successfully" mesajı görünecek

İşlem 30 saniye sürer

🔄 Alternatif Yöntemler

Purge Everything butonunu bulamazsanız:

  1. Quick Actions: Dashboard'da Quick Actions bölümünde
  2. Overview: Ana sayfada sağ üst köşede
  3. Custom Purge: Purge Custom Files seçeneği
  4. Development Mode: Geçici olarak açabilirsiniz

5Development Mode Alternatifi

Cache temizleme butonunu bulamazsanız:

Overview → Development Mode → ON

Bu 3 saat boyunca tüm cache'i bypass eder

⏱️ Bekleme Süresi

Cache temizledikten sonra:

  • 5 dakika bekleyin (global propagation)
  • Sayfayı yenileyin (Ctrl+F5)
  • Navigation linklerini test edin

🎉 Başarı Beklentisi

Cache temizlendikten sonra:

  • ✅ Page Rule etkili olacak
  • ✅ JavaScript navigation çalışacak
  • ✅ Site normal hızda olacak

6Son Kontrol

5 dakika sonra aşağıdaki linkleri test edin:

  • Ana sayfa linklerini tıklayın
  • Navbar menülerini test edin
  • JavaScript butonlarını kontrol edin
