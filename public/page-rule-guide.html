📋 Page Rule Cache Bypass - Final Solution

🎯 KESIN ÇÖZÜM: Page Rule

Bot Fight Mode ve Rocket Loader kapalıysa, Page Rule ile cache bypass yaparak tüm JavaScript optimizasyonlarını devre dışı bırakacağız.

1Rules Menüsüne Gitme

Cloudflare Dashboard'da sol menüden Rules sekmesine tıklayın

Rules → Page Rules

Ardından Page Rules alt menüsünü seçin

2Yeni Page Rule Oluşturma

Create Page Rule butonuna tıklayın

Not: Ücretsiz plan 3 Page Rule'a izin verir. Yeterli olacaktır.

3URL Pattern Girme

URL alanına aşağıdaki pattern'i girin:

www.finrasyo.com/*

Bu pattern tüm sayfalara uygulanacak

4Settings Ekleme

Aşağıdaki ayarları sırasıyla ekleyin:

1. Cache Level: Bypass
2. Security Level: Essentially Off
3. Rocket Loader: Off
4. Mirage: Off
5. Polish: Off

Her ayar için Add a Setting butonuna tıklayın

5Page Rule Kaydetme

Save and Deploy butonuna tıklayın

✅ Page Rule aktif oldu ve tüm optimizasyonlar devre dışı

6Cache Temizleme

Son olarak cache'i temizleyin:

Caching → Configuration → Purge Everything

Bu işlem 30 saniye sürer

⏱️ Etkili Olma Süresi: Page Rule oluşturulduktan sonra 2-3 dakika içinde etkili olacaktır.
