🎯 Yöntem 1: Drag & Drop (Önerilen)

En kolay yöntem! GitHub'a yükleme yapmadan direkt Netlify'a deploy.

1

Netlify Hesabı Açın

Adres: netlify.com

Sign up → Email ile kayıt olun

Netlify Ana Sayfa

Sign up → Email + Password → Verify Email

2

Build Folder Hazırlayın

Replit'te projenizi build edin:

# Replit Shell'de çalıştırın npm run build # dist/ klasörü oluşacak ls dist/

Sonuç: dist/ klasörü oluşacak - bu klasörü Netlify'e yükleyeceğiz.

3

Dist Klasörünü İndirin

Replit'ten dist klasörünü bilgisayarınıza indirin:

  1. Replit Files → dist klasörü
  2. Sağ tık → Download
  3. ZIP dosyası indirilecek
  4. ZIP'i açın, dist klasörünü çıkarın
4

Netlify'e Deploy

Netlify dashboard'a gidin:

📁 Want to deploy a new site without connecting to Git?

Drag and drop your site output folder here

Or, browse to choose files

Adım: İndirdiğiniz dist klasörünü bu alana sürükleyin!

5

Deploy Tamamlandı

Upload bitince:

🎉 Site deployed successfully!

Site URL: https://amazing-site-12345.netlify.app

Site settings → Change site name

Netlify size otomatik URL verecek. Site çalışıyor!

6

Custom Domain Bağlayın

www.finrasyo.com domain'ini bağlayın:

  1. Site settings → Domain management
  2. Add custom domain → www.finrasyo.com
  3. DNS records gösterilecek
  4. Cloudflare'de A record'u Netlify IP'ye yönlendirin
  5. SSL otomatik aktif olacak

⚙️ Build Settings

📝 Gerekli Files (Replit'te oluşturun)

netlify.toml (kök dizinde):

[build] publish = "dist" command = "npm run build" [[redirects]] from = "/*" to = "/index.html" status = 200 [build.environment] NODE_VERSION = "20"

⚠️ Environment Variables

Eğer API keys kullanıyorsanız:

  1. Site settings → Environment variables
  2. Add variable → VITE_STRIPE_PUBLIC_KEY
  3. Database URL'i backend için ayrı ayarlayın

🌐 Custom Domain Setup

Cloudflare DNS Settings

A Record:

Name: www | Content: 75.2.60.5 (Netlify IP)

CNAME Record:

Name: @ | Content: www.finrasyo.com

🔄 Yöntem 2: GitHub Integration

A

GitHub Repository Import

Eğer GitHub'a yükleme yaptıysanız:

  1. Netlify → New site from Git
  2. Connect to GitHub
  3. Repository seçin: finrasyo
  4. Build settings yapın
  5. Deploy site tıklayın

✅ Deploy Başarı İşaretleri

  • Site URL'i açılıyor
  • Navigation linkler çalışıyor
  • SSL sertifikası aktif (🔒)
  • Mobile responsive
  • Page reload navigation sorunsuz

🛠️ Backend vs Frontend

Netlify: Sadece frontend hosting

Backend için:

  • Railway.app (kolay backend hosting)
  • Vercel (full-stack)
  • Replit (mevcut backend'i kullan)

Database: Neon, PlanetScale, Supabase (ücretsiz)
