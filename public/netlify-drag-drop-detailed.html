1

Projeyi Build Edin

Replit Shell'de şu komutu çalıştırın:

~ $ npm run build

⚠️ Build Sırasında Dikkat Edilecekler

  • Internet bağlantınız stabil olmalı
  • Build işlemi 2-5 dakika sürebilir
  • Hata alırsanız önce npm install çalıştırın
  • Build tamamlandığında "Built in X seconds" mesajı görünür

Build başarılı olduğunda şunu göreceksiniz:

✓ built in 1.23s dist/index.html 12.34 kb dist/assets/index-abc123.js 456.78 kb dist/assets/index-def456.css 23.45 kb

Sonuç: dist/ klasörü oluşacak ve içinde web siteniz hazır olacak.

2

Dist Klasörünü Kontrol Edin

Replit Files panelinde dist klasörünü kontrol edin:

📂 Proje Klasör Yapısı

  • client/
  • server/
  • shared/
  • dist/ ← Bu klasör yeni oluştu
    • index.html
    • assets/
      • index-abc123.js
      • index-def456.css
      • finrasyo-logo.svg
  • package.json
  • netlify.toml

✅ Dist Klasörü Doğru İçerik

  • index.html - Ana sayfa dosyası
  • assets/ - CSS, JS, resim dosyaları
  • Toplam boyut: Yaklaşık 2-5 MB
3

Dist Klasörünü İndirin

Replit'ten bilgisayarınıza indirme adımları:

  1. Replit Files paneli açın (sol taraf)
  2. dist klasörüne sağ tıklayın
  3. "Download" seçeneğine tıklayın
  4. ZIP dosyası otomatik indirilecek (dist.zip)
  5. İndirilen ZIP dosyasını açın
  6. İçinden dist klasörünü çıkarın

🖼️ Replit Download İşlemi

Files panel → dist klasörü → Sağ tık → Download

Tarayıcınızın Downloads klasörüne dist.zip indirilecek

💾 İndirme Alternatifleri

Eğer sağ tık çalışmazsa:

  1. Dist klasörünü seç
  2. Üst menüde "..." → Download
  3. Ya da Ctrl+A → Tüm dosyaları seç → Download
4

Netlify Hesabı Oluşturun

Netlify'e kayıt olma adımları:

  1. netlify.com adresine gidin
  2. "Sign up" butonuna tıklayın
  3. Email + Password ile kayıt olun
  4. Email doğrulama linkine tıklayın
  5. Netlify dashboard'a giriş yapın

5

Drag & Drop Deployment

En önemli adım! Dikkatli takip edin:

📁 Netlify Deploy Alanı

Want to deploy a new site without connecting to Git?

Drag and drop your site output folder here

Or, browse to choose files

🎯 Drag & Drop İşlemi

  1. Bilgisayarınızda dist klasörünü bulun
  2. Klasörü TAM OLARAK Netlify'nin mavi alanına sürükleyin
  3. Klasörün içindekileri değil, klasörün kendisini sürükleyin
  4. Mouse'u bıraktığınızda upload başlayacak

✅ Upload Başladığında

Şu mesajları göreceksiniz:

  • "Uploading site files..." (Yükleniyor)
  • "Processing files..." (İşleniyor)
  • "Site deploy in progress..." (Deploy ediliyor)
  • "Site is live!" (Hazır!)
6

Deploy Tamamlandı

Deployment başarılı olduğunda:

https://amazing-butterfly-12345.netlify.app

Netlify size böyle rastgele bir URL verecek. Bu URL'e tıklayın!

🎉 Test Checklist

  • ✅ Site açılıyor mu?
  • ✅ Ana sayfa görünüyor mu?
  • ✅ Navigation linkler çalışıyor mu?
  • ✅ Hakkımızda, İletişim sayfaları açılıyor mu?
  • ✅ Mobil görünüm düzgün mü?

⚠️ Yaygın Sorunlar ve Çözümleri

  • 404 Error: dist klasörünün içini değil, klasörün kendisini yükleyin
  • Beyaz sayfa: Build işlemini tekrar yapın
  • CSS yok: Vite config'i kontrol edin
  • API çalışmıyor: Backend'i ayrı host etmek gerekir
7

Custom Domain (www.finrasyo.com)

Kendi domain'inizi bağlama adımları:

  1. Netlify site'ınızda "Site settings" tıklayın
  2. "Domain management" seçin
  3. "Add custom domain" butonuna tıklayın
  4. "www.finrasyo.com" yazın ve confirm edin
  5. DNS ayarları gösterilecek
# Cloudflare DNS ayarları A @ 75.2.60.5 CNAME www amazing-butterfly-12345.netlify.app

🔧 DNS Ayarlama

  1. Cloudflare dashboard'a gidin
  2. DNS → Records
  3. www.finrasyo.com A record'unu Netlify IP'ye yönlendirin
  4. 15-30 dakika bekleyin
  5. SSL otomatik aktif olacak
