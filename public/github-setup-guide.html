🚀 GitHub Otomatik Deployment Kurulumu

ADIM 1: GitHub Hesabı Oluşturun

  1. github.com adresine gidin
  2. "Sign up" butonuna tıklayın
  3. Email, kullanıcı adı ve şifre girin
  4. Email doğrulaması yapın
  5. Ücretsiz plan seçin (Free tier yeterli)

Zaten hesabınız varsa bu adımı atlayın

ADIM 2: Yeni Repository Oluşturun

  1. GitHub'da "New repository" butonuna tıklayın
  2. Repository name: finrasyo-website (veya istediğiniz isim)
  3. Description: FinRasyo Financial Analysis Platform
  4. Public seçin (ücretsiz)
  5. "Add README file" işaretleyin
  6. "Create repository" butonuna tıklayın

ADIM 3: Proje Dosyalarını Hazırlayın

  1. Bilgisayarınızda proje klasörünü açın
  2. Terminal/Command Prompt açın
  3. Proje klasörüne gidin
  4. Aşağıdaki komutları sırayla çalıştırın:
git init git add . git commit -m "İlk commit" git branch -M main git remote add origin https://github.com/KULLANICI_ADINIZ/finrasyo-website.git git push -u origin main

NOT: KULLANICI_ADINIZ yerine GitHub kullanıcı adınızı yazın

⚠️ Git Kurulu Değilse

Windows: git-scm.com'dan Git indirin

Mac: Terminal'de "git --version" yazın, otomatik kurulur

Linux: sudo apt install git (Ubuntu/Debian)

ADIM 4: Netlify'de Yeni Site Oluşturun

  1. app.netlify.com adresine gidin
  2. "Add new site" butonuna tıklayın
  3. "Import an existing project" seçin
  4. "GitHub" butonuna tıklayın
  5. GitHub hesabınızla giriş yapın ve izin verin
  6. Repository listesinden finrasyo-website seçin

ADIM 5: Build Ayarlarını Yapın

  1. Build command: npm run build
  2. Publish directory: dist
  3. Base directory: boş bırakın
  4. "Deploy site" butonuna tıklayın

Netlify otomatik olarak build alacak ve site deploy edilecek

ADIM 6: Domain Bağlantısı

  1. Deploy tamamlandıktan sonra "Site settings" gidin
  2. "Domain management" sekmesi
  3. "Add custom domain" tıklayın
  4. finrasyo.com yazın ve ekleyin
  5. "Verify DNS configuration" bekleyin

DNS zaten Cloudflare'de yapılmış, otomatik tanınacak

ADIM 7: Eski Site'ı Silin

  1. Netlify dashboard'da eski manuel site'ı bulun
  2. "Site settings" → "General"
  3. En aşağıda "Delete site" butonuna tıklayın
  4. Site adını yazıp onaylayın

✅ Kurulum Tamamlandı!

Artık şöyle çalışacak:

  • Kod değiştirip GitHub'a push yapın
  • Netlify otomatik build alır
  • 2-3 dakika sonra www.finrasyo.com güncellenir
  • Hata durumunda otomatik rollback

🧪 İLK TEST

Kurulumu test etmek için:

  1. Projede küçük bir değişiklik yapın
  2. Terminal'de şunları çalıştırın:
git add . git commit -m "Test değişikliği" git push

2-3 dakika bekleyip www.finrasyo.com kontrol edin

🎯 Sonuç

Artık her kod değişikliği otomatik olarak siteye yansıyacak!
