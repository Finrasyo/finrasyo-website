🧪 Test ve Domain Bağlama

Site testinden domain kurulumuna kadar

🔍 Bölüm 1: Site Testi

Upload sonrası Netlify size otomatik URL verecek:

https://amazing-butterfly-12345.netlify.app

✅ Test Checklist - Sırayla Kontrol Edin

  1. Ana Sayfa Test:
    • URL'e tıklayın
    • Site açılıyor mu?
    • FinRasyo başlığı görünüyor mu?
    • Sayfanın tasarımı düzgün mü?
  2. Navigation Test:
    • Navbar'daki "Hakkımızda" linkine tıklayın
    • Sayfa açılıyor mu? (404 yok mu?)
    • "İletişim" linkini test edin
    • "Nasıl Çalışır" linkini test edin
  3. Mobil Test:
    • Tarayıcıda F12 basın
    • Mobile view'e geçin
    • Site responsive mı?
    • Menü çalışıyor mu?
  4. Hız Test:
    • Sayfa hızlı yükleniyor mu?
    • Resimler görünüyor mu?
    • CSS stilleri yüklendi mi?

⚠️ Yaygın Test Sorunları

  • 404 Error: public klasörünü değil, içeriğini yüklemişsiniz
  • Beyaz sayfa: index.html bulunamıyor
  • CSS yok: assets klasörü eksik
  • Navigation 404: React router ayarları eksik

Çözüm: netlify.toml dosyası redirect ayarlarını içeriyor

🌐 Bölüm 2: Custom Domain (www.finrasyo.com)

Adım 1: Netlify Domain Ayarları

  1. Netlify site dashboard'ınızda "Site settings" tıklayın
  2. Sol menüden "Domain management" seçin
  3. "Add custom domain" butonuna tıklayın
  4. "www.finrasyo.com" yazın ve Enter'a basın
  5. "Yes, add domain" onaylayın

📋 Netlify Size DNS Bilgileri Verecek

Domain eklendikten sonra şu bilgileri göreceksiniz:

# Netlify DNS Ayarları CNAME: www.finrasyo.com → amazing-butterfly-12345.netlify.app A Record: finrasyo.com → 75.2.60.5

Adım 2: Cloudflare DNS Ayarları

  1. Cloudflare Dashboard'a gidin
  2. DNS → Records seçin
  3. Mevcut www A record'unu bulun
  4. Edit tıklayın
  5. Target: amazing-butterfly-12345.netlify.app
  6. Proxy status: DNS Only (gri bulut)
  7. Save tıklayın

🔧 Cloudflare DNS Ayar Detayları

Eski ayar (silinecek):

Type: A | Name: www | Content: 88.99.123.45 | Proxy: ON

Yeni ayar:

Type: CNAME | Name: www | Content: amazing-butterfly-12345.netlify.app | Proxy: OFF

Adım 3: SSL ve HTTPS

  1. DNS değişikliği 15-30 dakika sürer
  2. Netlify otomatik SSL sertifikası oluşturur
  3. "HTTPS certificate provisioned" mesajını bekleyin
  4. www.finrasyo.com test edin
  5. 🔒 simgesi görünmelidir

🎯 Bölüm 3: Final Test

🏆 Production Test Checklist

  1. Domain Test:
    • www.finrasyo.com açılıyor mu?
    • SSL aktif mi? (🔒 var mı?)
    • HTTP otomatik HTTPS'e yönlendiriliyor mu?
  2. Navigation Test:
    • Tüm navbar linkleri çalışıyor mu?
    • Sayfalar arası geçiş sorunsuz mu?
    • Tarayıcı back/forward butonları çalışıyor mu?
  3. Performance Test:
    • Sayfa yükleme hızı kabul edilebilir mi?
    • Tüm resimlerde loading animasyonu var mı?
    • Mobile görünüm optimize mi?
  4. SEO Test:
    • Sayfa başlığı doğru mu?
    • Meta description var mı?
    • Favicon görünüyor mu?

🎉 Başarı Kriterleri

Deployment başarılı sayılır eğer:

  • ✅ www.finrasyo.com domain'i açılır
  • ✅ Navigation problemi tamamen çözülür
  • ✅ SSL sertifikası aktif olur
  • ✅ Site hızlı yüklenir
  • ✅ Mobile responsive çalışır
  • ✅ Cloudflare navigation engeli ortadan kalkar

🚨 Sorun Yaşarsanız

  • DNS propagation: 24 saate kadar sürebilir
  • SSL pending: Cloudflare proxy'yi kapatın
  • 404 errors: netlify.toml dosyası eksik
  • Slow loading: Netlify CDN cache temizleyin
