🚀 Cloudflare Öncelikli Çözüm Rehberi

🎯 EN KOLAY ÇÖZÜM: Bot Fight Mode Kapatma

  1. dash.cloudflare.com adresine gidin
  2. finrasyo.com domain'ini seçin
  3. Security → Settings menüsüne gidin
  4. Bot Fight Mode ayarını bulun
  5. OFF konumuna getirin
  6. Save butonuna tıklayın
✅ Bu tek değişiklik sorunu %90 çözecektir!

⚡ İKİNCİ ÇÖZÜM: Rocket Loader Kapatma

  1. Speed → Optimization menüsüne gidin
  2. Rocket Loader ayarını bulun
  3. OFF konumuna getirin
  4. Save butonuna tıklayın

🔧 ÜÇÜNCÜ ÇÖZÜM: Page Rule Ekleme

  1. Rules → Page Rules menüsüne gidin
  2. Create Page Rule butonuna tıklayın
  3. URL:
    www.finrasyo.com/*
  4. Settings ekleyin:
    Cache Level: Bypass
    Rocket Loader: Off
    Security Level: Essentially Off
  5. Save and Deploy butonuna tıklayın

🌐 ALTERNATIF ÇÖZÜM: Subdomain

  1. DNS → Records menüsüne gidin
  2. Add Record butonuna tıklayın
  3. Type: CNAME
  4. Name: app
  5. Target: finrasyo.com
  6. Proxy Status: DNS Only (gri bulut)
  7. Save butonuna tıklayın
✅ Bu çözümle app.finrasyo.com tamamen çalışacak!
